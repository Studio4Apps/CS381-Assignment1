# 🎓 Personal Academic Portfolio - CS381

Welcome to my personal academic portfolio repository! This project is Assignment 1 for the **Web Application Development (CS381)** course.

## 🚀 Project Overview
This project is a fully responsive, semantic, and accessible personal website designed from scratch using only HTML5 and custom CSS3. It demonstrates a mobile-first approach without the use of any external CSS frameworks like Bootstrap or Tailwind, and strictly no JavaScript.

## ✨ Key Features
* **📱 Mobile-First Design:** Layouts are optimized for all screen sizes starting from mobile up.
* **🎨 Advanced CSS Layouts:** Heavy utilization of both **Flexbox** and **CSS Grid** for modern structural alignment.
* **📝 Semantic HTML5:** Proper use of semantic tags for better document structure and SEO.
* **♿ Accessibility:** Built with good accessibility considerations in mind.
* **📄 Multi-Page Structure:** The portfolio is neatly organized into 3 separate pages (Index, Profiles, and Contact).

## 📂 Repository Contents
This repository includes all the required deliverables for the assignment:
* `HTML` and `CSS` source code files.
* `Design_Explanation.pdf`: A 1-page report detailing the design choices and layout strategy.
* `Wireframes`: The initial sketches/wireframes used to plan the website layout.
